### Committer
说明：各个开源项目的核心成员即为Committer，其拥有所参与的开源项目的Push权限；
评选规则：由DOPMC Member和Committers匿名投票产生；

### Contributor
说明：各个开源项目的贡献者即为Contributor；
评选规则：为开源项目提交Pull request并被Merge；
